# Keyboard Mouse
This is a **simple** and **VERY** customizable **Python** program to control your **mouse** with a **keyboard**!

### HOW TO USE:
  * Open the **program**.
  * Select **inputs**. (Up, down, left, ect.)
  * Use your **special** keybutton to turn on and off the keyboard mouse controller.
  * That's **it**!

# NOTE
This **WILL** block whatever key you put (other then the **special** key you set), so if it acts haywire, just know, your **mouse STILL WORKS**! So if your **keyboard** starts acting **weird** or something, use your **mouse** to navigate to whatever you used the **script** in (Probably just a **terminal**) and **CLOSE IT**!
